# Regression Models Portfolio

Welcome to my **Regression Models Portfolio**! This repository contains implementations and analysis of several regression techniques in Python.

---

## **1. Linear Regression**
Linear Regression is a statistical method to model the relationship between a dependent variable and one independent variable.  
- **Purpose:** Predict continuous values.  
- **Example Use Case:** Predicting salary based on years of experience.  
- **Python Libraries Used:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`  

**Features Implemented:**
- Training and testing a linear regression model
- Visualizing regression line
- Calculating model performance metrics (R² score, Mean Squared Error)

---

## **2. Multiple Linear Regression**
Multiple Linear Regression extends Linear Regression to handle multiple independent variables.  
- **Purpose:** Predict a target variable based on multiple features.  
- **Example Use Case:** Predicting mouse Size using Tail,Weight of mouse.
 
- **Python Libraries Used:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`  

**Features Implemented:**
- Handling multiple features
- Model training and prediction
- Feature correlation analysis
- Performance evaluation

---

## **3. Polynomial Regression**
Polynomial Regression models the relationship between the independent variable and the dependent variable as an nth degree polynomial.  
- **Purpose:** Capture non-linear relationships between variables.  
- **Example Use Case:** Predicting growth curves or stock prices with non-linear trends.  
- **Python Libraries Used:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`  

**Features Implemented:**
- Transforming features into polynomial form
- Model training and prediction
- Visualizing polynomial regression curves
- Calculating model performance metrics
